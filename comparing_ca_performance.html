Logestic Regression Performance¶

- Create a Logistic Regression Model : 

    -> logreg_clf = LogisticRegression()

- Fit the Model with Training Set (X_train, Y_train) :

    -> logreg_model = logreg_clf.fit(X_train, Y_train)

- Predicting Y based on X_Test Set:

    -> logreg_prediction = logreg_clf.predict(X_test)

- Accuracy:

    -> 100*accuracy_score(logreg_prediction, Y_test)

- Confusion Matrix:

    -> confusion_matrix(logreg_prediction, Y_test)

- classification_report:

    -> classification_report(logreg_prediction, Y_test)
